About IDU - Interstellar Detect Unknown

Overview

The Interstellar Detection - Unknown (IDU) project is a collection of advanced astronomical visualization and simulation tools. This Flask web application provides a unified interface to explore various cosmic phenomena through interactive visualizations.

Available Visualizations

  • Star Mapper - Live Sky: Real-time star mapping using astronomical data
  • Quantum-Enhanced Sky Watcher: Advanced anomaly detection with quantum computing
  • Binary Transmitter: Visualization of binary signal transmission
  • Entropy-Driven Atlas Observation: Real telescope data analysis
  • Radio Waves Simulation: Quantum radio wave propagation
  • Solar Defense: Solar activity monitoring
  • 3D Atlas: Three-dimensional cosmic visualization

Technologies Used

  • Python: Core programming language
  • Flask: Web framework for the application interface
  • Pygame: Real-time graphics and visualization
  • OpenCV: Computer vision and image processing
  • NumPy & Pandas: Data analysis and manipulation
  • Skyfield: Astronomical calculations
  • QuTiP: Quantum computing simulations

How to Use

  1. Navigate to the Home page
  2. Select a visualization from the available options
  3. Click "Launch Visualization" to view details
  4. Click "Run Visualization" to start the application
  5. The visualization will open in a separate window
  6. Use "Stop Visualization" to close the application

Requirements

All Python dependencies are listed in requirements.txt. Install them using: pip install -r requirements.txt
